COLORES
Formatos Descripción Herramienta
Nombres Podemos especificar un color usando un nombre predefinido. (En inglés) css - color
Modelo RGB RGB Podemos establecer el valor de un color mezclando diferentes intensidades de los tres colores primarios. (Red, green, blue)
RGBA Similar al RGB, pero con un parámetro adicional denominado opacidad (transparencia). (Red, green, blue, opacity)
Valor hexadecimal Podemos utilizar un número hexadecimal para definir un color. #hexadecimal
Modelo HSL HSL En este modelo utilizamos conceptos que son más fáciles de comprender (hue, saturation, lightness). (tono,saturación,luminosidad)
HSLA Esta función es idéntica a la función hsl, pero con un parámetro adicional denominado opacidad. (h,s,l,opacity)
SELECTORES
Tipo Definición Sintaxis
Universal Selecciona todos los elementos. * { }
De tipo Permite seleccionar elementos usando el nombre del elemento a { ]
De clase Permite seleccionar elementos en función de los valores del atributo class de los elementos. .valor_atributo { }
De ID Permite seleccionar un elemento en función del valor del atributo ID del elemento. #valor_atributo { }
De atributo Permite seleccionar elementos en función de la presencia o del valor de un determinado atributo. a[title] { }
Pseudo-clases Permite seleccionar elementos en base a su estado (:disabled), posicion de los elementos en el documento(:nth-child) e interacción del usuario(:hover). Pseudo-clases
Pseudo-elementos Seleccionan parte de un elemento en lugar del elemento mismo. Pseudo-elementos

Características
CAJAS EN BLOQUE Ocupan todo el espacio disponible en su contenedor aunque sus contenidos no lleguen al final de la línea.
Siempre comienzan en una nueva línea.
Las propiedades width y height se respetan.
Las propiedades padding, margin y border harán que el contenido que rodea a la caja, se aleje de esta.
CAJAS EN LÍNEA Estas cajas solo ocupan el espacio necesario para mostrar su contenido. No empiezan en una nueva línea.
Las propiedades width y height no se respetan
Las propiedades padding, margin y border verticales se aplican, pero no harán que el contenido que rodea a la caja en la parte superior e inferior, se aleje de esta.
Las propiedades padding, margin y border horizontales se aplican, y harán que el contenido que rodea a la caja en la parte izquierda y derecha, se aleje de esta.
ENFOQUE MIXTO No comienzan en una nueva línea.
Las propiedades width y height se respetan
Las propiedades padding, margin y border harán que el contenido que rodea a la caja, se aleje de esta.
CONSULTA DE MEDIOS
Una media query consiste básicamente en una media type (es decir un tipo de dispositivo), y una o mas expresiones que contienen características del media type, por ejemplo ancho de la ventana del navegador, que pueden ser evaluados a true o false.
Ejemplo:
            @media screen and (max-width: 480px) {
                body{
                    background-color: green;
                }
            }
El ejemplo básicamente dice que si el media type es una pantalla y el tamaño máximo de la ventana del navegador es igual o menor a 480px, hacer del color de fondo de elemento body verde.

Sintaxis

MEDIA FEATURE
Feature Descripción
width Usada para verificar el ancho exacto de la ventana del navegador.
max-width Usada para verificar el ancho mas pequeño de la ventana del navegador (viewport).
min-width Usada para verificar el ancho mas grande de la ventana del navegador (viewport).
height Usada para verificar el alto exacto de la ventana del navegador.
max-height Usada para verificar el alto mas pequeño de la ventana del navegador (viewport).
min-height Usada para verificar el alto mas grande de la ventana del navegador (viewport).
aspect-ratio Usada para verificar la relación ancho/alto del viewport
orientation Usada para verificar la orientación de la pantalla( portrait / landscape)
resolution Usada para verificar la densidad de pixeles del dispositivo.
grid Usada para verificar si el dispositivo usa una pantalla basada en grid o bitmap.
color Usada para verificar el numero de bits por componente de color (red, green, blue) del dispositivo.
